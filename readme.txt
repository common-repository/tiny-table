== Description ==

The Tiny Table plugin re-adds table support to the TinyMCE Rich Visual Editor.
The TinyMCE table plugin is normally bundled with TinyMCE, but it has be removed from the
version within WordPress.  This plugin puts it back in.

Most of the credit for this plugin belogns to [Moxiecode Systems] (http://tinymce.moxiecode.com/ ),
who wrote the Table plugin for TinyMCE, which this plugin uses.

== Installation ==

1. copy the 'tiny-table' directory to your 'wp-contents/plugins' directory.
1. Activate the Tiny Table in your plugins administration page.
1. You will now see table button when using the the Rich Visual Editor, TinyMCE.
